## 기능 요구사항

- 질문은 hard delete가 아닌 soft delete로 delete 상태값 변경
- 로그인 사용자와 질문한사람이 같은 경우만 삭제 가능
- 답변이 없는 경우 삭제가능
  - 답변이 있어도 모든 답변의 답변자가 질문자와 동일한경우 삭제가능
  - 질문을 삭제할 때 답변도 삭제해야하며 답변 또한 soft delete
- 질문자와 답변자가 다르면 삭제할수 없음
- 질문과 삭제 이력 정보는 DeleteHistory를 활용
